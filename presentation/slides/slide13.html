实战经验与踩坑总结

Agent 是一门"手艺"——效果极度依赖使用者的 prompt 能力和对工具的理解

🔧
手艺与半成品
  • 加功能很爽，但微调按钮位置/布局/交互时往往改不对
  • 了解 AI 能力边界 + prompt engineering 是核心技能
  • Agent 越用越好用——需要在团队内持续积累经验
💰
成本与投入
  • 一个任务 $5-10 是常态；复杂任务 $20+ 迅速蒸发
  • "穷人思维"反而限制了探索；但盲目跑也会烧钱
  • 写好 AGENT.md / Cursor Rules，团队内积累 prompt 和知识库
🔄
心态与工作流
  • 把 Agent 当外包 / 初级工程师：派活 → 检查 → 给反馈
  • 最佳实践：Agent 生成初版 → 人工/Cursor 精修
  • 不要期望一次性搞定，迭代式协作效果更好
"Still early, but already useful (if used correctly)" — 所有工具都处于这个阶段。未来能把 Agent 用得最好的，大概率还是专业人士。
