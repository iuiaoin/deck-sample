1
半自动 → 全自动
从 AI-assisted coding 向全自动 Agent 进化是明确方向。Agent 能独立工作的时间越来越长，做事的数量和质量持续提升。
2
IDE 与 CLI 双轨并行
Cursor 要做 All-in-one 入口，Claude Code 证明 IDE 非必须。程序员自己写代码时仍会回归传统 IDE，Agent 工作无需 IDE。
3
成本是最大瓶颈
LLM 降价将重塑格局。原厂做 agent 有闲时资源优势。性能 vs 成本的 tradeoff 是 agent 公司的核心决策。
4
壁垒的本质是"快"
在这个发展过快的领域，任何技术领先都可能被迅速追赶。一旦慢下来，就可能被超越。Cursor 创始人的残酷认知。
